# Constant D(x,y) function

See the following paper https://doi.org/10.3390/fractalfract6020074.

## Usage

``` r
.constant_d(x, y)
```

## Arguments

- x:

  `Float (positive)`. First argument of the function.

- y:

  `Float (positive)`. Second argument of the function.

## Value

A positive `Float` corresponding to the value the function evaluate at
(x,y).
