# Epanechnikov kernel function

Epanechnikov kernel function

## Usage

``` r
epanechnikov(u)
```

## Arguments

- u:

  `numeric`. Scalar or vector of numeric values at which to evaluate the
  function.

## Value

A scalar or vector of `numeric`.
